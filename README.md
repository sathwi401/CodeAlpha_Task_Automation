# 🤖 Task Automation with Python

## 📌 Project Overview

This project is developed as part of the CodeAlpha Python Programming Internship. It automates common real-world tasks using Python.

## ✨ Features

### 📁 Move JPG Files

* Move all `.jpg` files from one folder to another.
* Automatically creates the destination folder.
* Prevents overwriting existing files.

### 📧 Email Extraction

* Extract email addresses from a text file.
* Remove duplicate email addresses.
* Save extracted emails to a new file.

### 🌐 Webpage Title Scraper

* Fetch the title of any webpage.
* Optionally save the result to a text file.

## 🛠 Technologies Used

* Python 3

## 📚 Python Concepts

* File Handling
* Regular Expressions
* Functions
* Loops
* Conditional Statements
* os Module
* shutil Module
* HTML Parsing

## ▶️ How to Run

1. Download the Python file.
2. Open a terminal.
3. Run:

python task_automation.py

4. Select an option from the menu.
